# Event Planning Helper 🎉

## Overview

Event Planning Helper is an AI-powered application built with PartyRock. The app helps users organize and plan events by generating a detailed event plan, budget breakdown, and invitation message based on user inputs.

## Features

* Generate a complete event plan
* Create a budget allocation breakdown
* Generate customized invitation messages
* Easy-to-use input fields
* Fast and repeatable AI-powered recommendations

## Inputs

* Event Type
* Number of Guests
* Budget
* Event Date

## Outputs

### Event Plan

Provides a detailed plan including:

* Preparation timeline
* Venue suggestions
* Food and beverage planning
* Entertainment ideas
* Event checklist

### Budget Breakdown

Provides:

* Budget allocation by category
* Estimated costs
* Percentage distribution

### Invitation Message

Generates a professional and engaging invitation message for guests.

## Technologies Used

* PartyRock
* Amazon Bedrock
* Generative AI

## Purpose

This project was created to demonstrate how generative AI can be used to build reusable applications that provide a structured and consistent user experience compared to traditional chatbot interactions.
